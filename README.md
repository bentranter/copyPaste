# Copy-Paste Proof of Concept

Test to see if copy-pasting onto a website works form a Chrome extension.

### Installation & Usage

1. Clone this repo
2. Open Chrome
3. Visit the special URL, `chrome://extensions`
4. Enable developer mode
5. Click "Load unpacked extension"
6. Select this extension
7. Visit the metalab.co site
8. Try to paste an image on the home page.

It should paste the image into the main section above the fold.
